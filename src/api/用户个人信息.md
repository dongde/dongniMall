# 川料智选——用户个人信息
### 接口功能

获取用户信息

### URL地址

```javascript
http://localhost:8081/user/queryUserByPhone
```

### 请求方式

`GET`

### 请求参数

| 参数      | 说明                               | 类型      | 可选值       | 备注    |
|---------- |---------------------------------- |---------- |------------- |-------- |
|phone | 用户手机号 | String | 必填 | |

### 请求参数体

无

### 接口示例

`http://localhost:8081/user/queryUserByPhone?phone=15680479106`

### 返回示例

```json
{
    "status": 200,
    "msg": "OK",
    "data": {
        "phone": "15680479106",
        "name": null,
        "gender": null,
        "address": null,
        "email": null,
        "postal_code": null,
        "password": null,
        "create_time": null
    },
    "ok": null
}
```
