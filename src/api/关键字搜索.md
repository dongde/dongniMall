# 川料智选 ——新闻资讯



### 接口功能

搜索新闻关键字

### URL地址

```
http://localhost:8081/article/search
```

### 请求方式

`GET`

### 请求参数

|  参数  |   说明   |  类型   | 可选值 | 默认值 |       备注       |
| :----: | :------: | :-----: | :----: | :----: | :--------------: |
|  page  | 当前页数 | Integer | 非必填 |   1    |     分页所需     |
| limit  | 每页条数 | Integer | 非必填 |   10   | 分页返回数据条数 |
| key  |   搜索关键字   | String  | 非必填 |  null  |   查询数据条件   |

### 接口示例

```
http://localhost:8081/article/search?page=1&limit=3&key=火锅
```
