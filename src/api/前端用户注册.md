# 川料智选——前端用户注册
### 接口功能

添加用户信息

### URL地址

```javascript
http://localhost:8081/front/register
```

### 请求方式

`POST`

### 请求参数

无

### 请求参数体

| 参数      | 说明                               | 类型      | 可选值       | 备注    |
|---------- |---------------------------------- |---------- |------------- |-------- |
|phone  | 手机号 | String | 必填 | |
|name  | 用户昵称 | String | 必填 | |
|gender | 性别 | Integer | 必填 | 0男，1女 |
|address  | 收货地址 | String | 必填 | |
|email | 邮箱 | BigDecimal | 必填 | |
|postal_code  | 邮政编码 | String | 必填 | |
|password  | 密码 | String | 必填 | |

### 接口示例

`http://localhost:8081/front/register`

### 返回示例

```json
{
    "status":200,
    "msg":"OK",
    "data":null,
    "ok":null
}
```
