# 川料智选 ——收货人信息删除

### 接口功能

删除收货人信息

### URL地址

```
http://localhost:8080/user/removeRecInfo
```

### 请求方式

`POST`

### 请求参数

|   参数    |   说明   |     类型      | 可选值 |     备注     |
| :-------: | :------: | :-----------: | :----: | :----------: |
| id | 收货人信息id |    String     |  必填  |     |

### 接口示例

```
http://localhost:8080/user/removeRecInfo
```



### 返回示例

```
{
    "status":200,
    "msg":"OK",
    "data":null,
    "ok":null
}
```
