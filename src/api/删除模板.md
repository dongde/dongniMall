# 川料智选 ——删除模板



### 接口功能

删除模板

### URL地址

```
http://localhost:8081/template/delete
```

### 请求方式

`GET`

### 请求参数

| 参数 | 说明 |  类型  | 可选值 |      备注      |
| :--: | :--: | :----: | :----: | :------------: |
|  id  |  id  | String |  必填  | 通过id删除数据 |

### 接口示例

```
http://localhost:8081/template/delete
```



### 返回示例

```
{
    "status":200,
    "msg":"OK",
    "data":null,
    "ok":null
}
```
