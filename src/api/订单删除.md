# 川料智选——订单删除
### 接口功能

删除订单信息

### URL地址

```javascript
http://localhost:8081/order/removeOrder
```

### 请求方式

`POST`

### 请求参数

无

### 请求参数体

| 参数      | 说明                               | 类型      | 可选值       | 备注    |
|---------- |---------------------------------- |---------- |------------- |-------- |
|order_number  | 订单号 | String | 必填 | |

### 接口示例

    `http://localhost:8081/order/modifyOrder`

### 返回示例

```json
{
    "status":200,
    "msg":"OK",
    "data":null,
    "ok":null
}
```
