# 川料智选——购物车添加
### 接口功能

用户购物车添加功能

### URL地址

```javascript
http://localhost:8081/order/addGoods
```

### 请求方式

`POST`

### 请求参数

无

### 请求参数体

| 参数      | 说明                               | 类型      | 可选值       | 备注    |
|---------- |---------------------------------- |---------- |------------- |-------- |
|user_phone  | 用户账号 | String | 必填 | |
|goods_price  | 商品单价 | BigDecimal | 必填 | |
|goods_name | 商品名称 | String | 必填 | |
|goods_img  | 商品图片 | String | 必填 | |
|goods_count  | 商品数量 | String | 必填 | |
|subtotal  | 商品小计 | BigDecimal | 必填 | |

### 接口示例

    `http://localhost:8081/order/addGoods`

### 返回示例

```json
{
    "status":200,
    "msg":"OK",
    "data":null,
    "ok":null
}
```
