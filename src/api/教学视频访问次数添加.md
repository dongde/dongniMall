# 川料智选——教学视频访问次数添加
### 接口功能

添加教学视频访问次数

### URL地址

```javascript
http://localhost:8081/menu/addTeachVideoCount
```

### 请求方式

`POST`

### 请求参数

无

### 请求参数体

| 参数      | 说明                               | 类型      | 可选值       | 备注    |
|---------- |---------------------------------- |---------- |------------- |-------- |
|id | 教学视频ID | String | 必填 | |

### 接口示例

`http://localhost:8081/menu/addTeachVideoCount`

### 返回示例

```json
{
    "status":200,
    "msg":"OK",
    "data":null,
    "ok":null
}
```
